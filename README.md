# 系统建模
对于机器人的平衡，纵向以及转向运动，可以将模型简化为三维空间中的类一阶倒立摆模型，即忽略腿长的变化，转向运动由两个驱动轮差速运动控制。
## 模型定义
称机器人机体质心到两个驱动轮转轴中心的连线为摆杆，得到如图1所示平衡车模型。
